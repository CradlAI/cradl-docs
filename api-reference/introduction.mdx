## Base URL

Cradls’s API is built on REST principles and is served over HTTPS. To ensure data privacy, unencrypted HTTP is not supported.

The Base URL for all API endpoints is:

```
https://auth.lucidtech.ai 
```

## Authentication

Authentication to Cradl’s API is performed via the Authorization header with a Bearer token. To authenticate, you need to include the Authorization header with the word Bearer followed by your API key in your requests like so:

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
